# 요구사항 정리
## 기능 요약
* 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급한다.
* 당첨 번호를 입력하면 당첨 통계를 출력한다.

## InputView
- [X] 구입 금액을 입력받아 전달
- [X] 당첨 번호를 입력받아 전달
- [ ] 보너스 번호를 입력받아 전달

## Prize
- [X] 로또 등수, 당첨 번호 개수, 당첨 금액을 저장
- [ ] 당첨됐는지 판단 후 등수를 전달
  - 1, 3~5 등
  - 2등

## Lotto
- [X] 자동으로 로또를 생성
  - 로또는 1~45 이내의 6가지 숫자를 가짐
- [X] 로또가 몇 등인지 판단

## LottoGroup
- [x] 로또 리스트를 저장
- [x] 로또 내에 1~5등 당첨이 몇 개 나왔는지 판단

## LottoMachine
- [X] 로또를 구매하고 받은 당첨 번호로 당첨 통계를 출력

## OutputView
- [x] 로또 그룹 내의 모든 로또를 출력
- [x] 당첨 통계를 출력
