# 요구사항 정리
## 기능 요약
* 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급한다.
* 당첨 번호를 입력하면 당첨 통계를 출력한다.

## InputView
- [ ] 구입 금액을 입력받아 전달
- [ ] 당첨 번호를 입력받아 전달

## Lotto
- [ ] 자동으로 로또를 생성
  - 로또는 1~45 이내의 6가지 숫자를 가짐
- [ ] 입력받은 값과 로또 값이 같은지 비교

## LottoGroup
- [ ] 로또 리스트를 저장
- [ ] 로또 내에 1~5등 당첨이 몇 개 나왔는지 판단

## LottoMachine
- [ ] 로또를 구매하고 받은 당첨 번호로 당첨 통계를 출력

## OutputView
- [ ] 로또 그룹 내의 모든 로또를 출력
- [ ] 당첨 통계를 출력
